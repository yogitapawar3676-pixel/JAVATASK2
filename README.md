# JAVATASK2
This is a simple Python program.
It takes details from the user using the keyboard and shows them on the screen.


What this program does
Takes name from the user
Takes age
Takes height
Checks if the user is a student
Displays all the details

Concepts Used
Variables
Data types (number, decimal, text, true/false)
Console input
Print output

How to run
Open terminal or command prompt
Go to the project folder
Run the program using:

python main.py
